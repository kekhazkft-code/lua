## PÁRATARTALOM-ELSŐDLEGES Szabályozási Filozófia
A rendszer a PÁRATARTALMAT helyezi előtérbe a hőmérséklettel szemben mert:
  - Túl száraz → VISSZAFORDÍTHATATLAN termékkárosodás (felület repedés, kéregképződés)
  - Túl párás → Penészesedési kockázat, kritikus gyorsan kezelni
  - Túl hideg → Csak lassítja a folyamatot, TELJESEN HELYREÁLLÍTHATÓ
  - A termék egyensúlya az Abszolút Páratartalomtól (AH) függ, nem a hőmérséklettől

================================================================================
                    KÜLSŐ LEVEGŐ HASZNÁLATI STRATÉGIA
================================================================================

**Külső levegő CSAK hűtésre használható, SOHA párátlanításra**

A külső levegő nem tudja eltávolítani a nedvességet - a párátlanítás hideg vizes
hőcserélő tekercset igényel a kondenzációhoz. A bypass-nak ZÁRVA kell lennie
párátlanítás közben.

Döntési logika:
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                            │
│  Kell hűteni?  ──NEM──►  Nincs külső levegő                               │
│       │                                                                    │
│      IGEN                                                                  │
│       │                                                                    │
│       ▼                                                                    │
│  Kell párátlanítani?  ──IGEN──►  CSAK vízhűtés (0°C, bypass ZÁRVA)        │
│       │                                                                    │
│      NEM (csak hűtés)                                                      │
│       │                                                                    │
│       ▼                                                                    │
│  Kamra - Külső ≥ 5°C?  ──IGEN──►  KÜLSŐ LEVEGŐ használata (ingyenes hűtés)│
│       │                                                                    │
│      NEM                                                                   │
│       │                                                                    │
│       ▼                                                                    │
│  Vízhűtés (8°C, bypass NYITVA)                                            │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

| Feltétel | Vízhűtés | Külső Levegő | Bypass | Miért |
|----------|----------|--------------|--------|-------|
| Párátlanítás | IGEN (0°C) | NEM | ZÁRVA | Hideg tekercs kondenzál nedvességet |
| Hűtés, külső 5°C+-szal hidegebb | NEM | IGEN | NYITVA | Ingyenes hűtés |
| Hűtés, külső nem elég hideg | IGEN (8°C) | NEM | NYITVA | Mechanikus hűtés kell |
